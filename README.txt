		Merci d'utiliser ce programme !

N'hésitez pas à faire un tour dans le menu touches et explication avant de commencer.

;************************************

ATTENTION : Ce programme ne marche pas sur les versions postérieures à Windows XP (le mode plein écran n'est plus supporté)
Si vous êtes dans ce cas, vous pouvez utiliser DOSBox pour le faire fonctionner (http://www.dosbox.com/download.php?main=1)
(Téléchargez dosbox, puis faites glisser Pixel.com sur dosbox.exe)
Avantages de DOSBox :
- permet d'utiliser le DOSBox mode, qui utilise une interruption non disponible sous Windows Xp, qui permet de fixer une pause plus petite et donc d'augmenter la vitesse.
- marche sous Vista, Seven et même Linux (et sûrement Mac OS, j'ai pas essayé)
Inconvénients de DOSBox :
- Pas de mode plein écran : c'est tout pitit !
- Bugs lors de l'utilisation des fichiers, notamment pour l'éditeur de niveau qui plante si vous le chargez 3 fois (n'hésitez pas à recharger le programme)

;************************************

 Informations utiles sur le programme :

système de points :
-1 point par tir rouge ou bombe
+20 points par ennemi tué
+2 points par munitions restantes à la fin d'un niveau
+5 points par bomes restantes à la fin d'un niveau
Les munitions ou bombes non récupérées ne rapportent pas de points.


;***************************************************

Utilisation de l'éditeur de niveau :
Cliquez sur les différents éléments de la barre en haut de l'écran afin de pouvoir les ajouter dans le niveau.
- Le vaisseau ne peut se placer qu'un seule fois (logique).
- Les lignes bleues à droites, ce sont des murs (si si !). Ils ne peuvent se placer qu'horizontalement ou verticalement.
  Cliquez une première fois pour fixez le point de départ du mur, une deuxième fois pour l'arrivée.
  Vous pouvez être approximatif sur le deuxième clic, de toute façon le programme calcule si c'est un mur horizontal ou vertical et garde les mêmes coordonnées verticales/horizontales que pour le premier clic.
- Vous pouvez augmenter/diminuer les stocks de base en vies/munitions/bombes en cliquant sur le nombre à côté de leur icône respective.
  Un clic gauche augmente le stock, un clic droit le diminue.


Créez votre niveau puis envoyez-moi le code (créé dans TEMP_EDI.TXT lorsque vous quittez l'éditeur) afin que je le compile et l'ajoute au programme (et soyez heureux d'avoir participé à son enrichissement !)
